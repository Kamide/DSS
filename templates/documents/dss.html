What is DSS?

The Document Sharing System (DSS) is a web application that allows registered users to create, edit, and read each other's documents. The system is aimed at providing as satisfying of an experience for users as possible. To do this, this system has established the following features/guidelines.

Documents

As the core aspect of this system, the documents belonging to DSS have the following features:

  • Varying privacy levels
  • Access older versions of documents
  • Censor words in a universal taboo list
  • Search keywords within document
  • Ability to lock/unlock documents
  • Invite other users to work on documents
  • Formatting options that even allows inclusion of images

Users

Users that are registered under DSS are divided into different categories each with different levels of privileges. The following subsection lists the privileges of each type of user. For a more detailed description of each privilege, visit:

Guest User (GU)

Guest users have access to the least amount of privileges from the pool of all possible privileges. Their privileges are as follows:

  • Read documents and retrieve older versions of them
  • File complaints about documents
  • Suggest taboo words to an SU
  • Apply to be promoted to an OU

Ordinary User (OU)

Ordinary users have access to all privileges of a GU as well as the following:

  • Create a new document with any privacy level
  • Invite other OUs to work on one's document and accept/deny invites to work on documents
  • Lock a shared document to update it
  • Unlock a locked document
  • File complaints about updates made by other contributors or about the owner of a document
  • Handle complaints regarding contributors of one's document
  • Search keywords within one's document
  • Search information about other OUs based on username/interests

Super User (SU)

Super users have access to all privileges of an OU and thus all privileges in general. Their unique privileges are as follows:

  • Update the membership of individual users
  • Maintain the list of taboo words
  • Unlock any locked document
  • Handle complaints about an OU

Messages

The messaging system allows users to communicate with each other and exercise their privileges. Messaging is used to:

  • Suggest taboo words
  • File complaints
  • Make suggestions about documents
  • Invite users to work on shared documents
  • Engage in ordinary conversations
